# SHU-Beamer-Theme

本项目基于THU-Beamer-Theme稍作修改，用于上海大学计算机学院的日常报告用途。
改动内容包括：

- 主题颜色更改
- logo更改
- 画面比例更改
  
原项目地址：https://github.com/tuna/THU-Beamer-Theme
